Створити словник власних визначень за допомогою Generic function. Всередині має бути визначення для 3 властивостей - ключ, значення, опис (різних типів даних). Для отримання або запису використовувати get/set для реалізації доступу. Також для полів потрібно використовувати модифікатори доступу (на Ваш вибір). У результаті повинен бути словник термінів, який приймає на вхідний параметр різних типів даних для реалізації.
